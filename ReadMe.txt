Versão Definitiva: "calculadoraBeta"
    * versão do calculadoraAlfa com a implementação de TRY CATCH / THROW NEW 
        - A divisão por 0 com pontos flutuantes resulta em infinito;
        - Foi necessário o uso de "throw new" para lançar a exeção na execução do programa;


Protótipo: "calculadoraAlfa"
    * versão da calculadora aplicando métodos e verificação por if else

Primeiras versões para estabilizar o algoritmo: 
    CalcSimples e CalculadoraV2
